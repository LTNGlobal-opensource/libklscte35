These scripts exercise a number of use cases:

klscte35_parse: SCTE-35 to SCTE-104 to SCTE-35, then compare to make sure we got the same as input
klscte35_scte104: SCTE-35 to SCTE-104, then prints result of conversion
klscte35_scte104to35: SCTE-104 to SCTE-35, then prints result of conversion
klscte35_tojson: SCTE-35 to JSON, then prints result of conversion
